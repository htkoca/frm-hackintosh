Benvenuti in Pacifist!

Per iniziare potete fare una delle seguenti cose:

  1. Se volete cercare di reinstallare un file, ma non sapete da quale package è stato installato, potete usare l’opzione “Cerca receipt”.
  2. Se volete reinstallare file che sono parte della vostra installazione di macOS, fate clic sul pulsante “Apri installer Apple”. Se la vostra copia di macOS proviene da un mezzo fisico come un DVD o una penna USB, assicuratevi di inserirlo prima di premere il pulsante (se avete più di un disco, provate per primo il disco 1).
  3. Se volete ispezionare o estrarre file da un altro package o archivio, fate clic sul pulsante “Apri package…” per scegliere un file .pkg, .zip, .tar o .dmg da cui estrarre i file.

Se avete bisogno di aiuto, scegliete “Aiuto di Pacifist” dal menu Aiuto.
