Bienvenue dans Pacifist !

Pour commencer, vous pouvez faire l’une des choses suivantes :

  1. si vous souhaitez réinstaller un fichier mais ne savez pas par quel paquet il a été installé, vous pouvez utiliser la fonction “Recherche de reçus”.
  2. Si vous souhaitez réinstaller des fichiers faisant partie de votre installation macOS, cliquez sur le bouton “Ouvrir les installateurs Apple”. Si votre copie de macOS est sur un medium physique tel qu’un DVD ou une clé USB, assurez-vous de l’insérer avant de cliquer sur le bouton “Ouvrir les installateurs Apple” (si votre copie de macOS est sur plus d’un disque, essayez d’abord le disque 1).
  3. Si vous souhaitez inspecter ou extraire des fichiers d’un autre paquet ou archive, cliquez sur le bouton “Ouvrir un paquet” pour choisir un fichier .pkg, .zip, .tar, ou .dmg d’où extraire les fichiers.

Si vous avez besoin d’aide, veuillez choisir “Aide Pacifist” du menu aide.
